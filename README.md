# 🧠 Brainy

> Tu segundo cerebro. Anotá tareas y eventos, sacate el peso de la cabeza.

App web personal para gestionar tareas y eventos con countdown, optimizada para iPhone/iPad.

---

## ¿Qué es?

Brainy es una agenda personal minimalista para anotar tareas rápidas del día a día
(parciales, finales, recados, pendientes de Globant) y eventos con cuenta regresiva
de días. Pensada para usarse desde el home screen del iPhone/iPad como si fuera una app nativa.

## Stack

- **Frontend:** HTML/CSS/JS standalone (un solo archivo `index.html`)
- **Backend:** Google Apps Script (API) + Google Sheets (base de datos)
- **Hosting:** GitHub Pages
- Sin frameworks, sin build steps, sin servidores propios.
